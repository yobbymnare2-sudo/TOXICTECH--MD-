### üöÄ `Deployment Guide`

**TOXICTECH-MD** requires a Session ID to run. Follow the steps below to get yours and deploy the bot.

#### `1Ô∏è‚É£ Fork the Repository`
Click the **Fork** button at the top right of this page to create a copy of this repository in your account.

#### `2Ô∏è‚É£ Get Session ID`
1. Click the button above to visit the **Pairing Site**.
2. Enter your WhatsApp number (with country code, e.g., `9476...`).
3. Click **GENERATE**.
4. Copy the **Session ID** (starts with `Toxicyobby=`).

#### `3Ô∏è‚É£ Deploy on Bot-Hosting Platforms`

<details>
<summary><b>üì¶ Deploy on Bot-Hosting.net (Recommended)</b></summary>

1. Go to [bot-hosting.net](https://bot-hosting.net) and create an account.
2. Create a new server and select **Node.js**.
3. Upload the ZIP file of this repository.
4. Go to the **Environment Variables** tab.
5. Add the following:
   - **Variable Name:** `SESSION_ID`
   - **Value:** `Your_Session_ID_Here`
6. Start the server.

</details>

<details>
<summary><b>üåê Deploy on Render (Alternative)</b></summary>

1. Go to [Render.com](https://render.com).
2. Create a **New Web Service** and connect your GitHub repository.
3. Set the following:
   - **Environment:** `Node`
   - **Build Command:** `npm install`
   - **Start Command:** `npm start`
4. Add the Environment Variable `SESSION_ID` in the dashboard.
5. Click **Deploy**.

</details>
